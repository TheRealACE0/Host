# Dependencies
- [MongoDB](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-debian/)
- [Docker](https://www.docker.com/)

# How to install 
1. git clone https://github.com/game-server-panel/game-server-panel
2. cd game-server-panel
3. npm dependencies
4. npm start
